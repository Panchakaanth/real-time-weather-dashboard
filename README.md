# 🌤 Real-Time Weather Dashboard

A dynamic weather dashboard built using **HTML**, **CSS**, and **JavaScript** that fetches and displays real-time weather data from a public REST API using asynchronous JavaScript.

---

## 🚀 Features

- Fetch live weather data using Fetch API
- Uses async/await for asynchronous operations
- Search weather by city name
- Dynamic DOM updates
- Real-time temperature, humidity, and wind speed
- Weather condition icons
- Comprehensive error handling
- Responsive and clean UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Async/Await
- REST API
- WeatherAPI.com
